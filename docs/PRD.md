# PRD: Stock Dashboard (통합 주식 대시보드)

## 1. 제품 개요

### 1.1 목적
미국 주식 시장(NYSE/NASDAQ)을 한눈에 파악하고, 실제 포트폴리오를 관리하며, 시장 심리를 AI로 분석하는 통합 웹 대시보드.

### 1.2 타겟 유저
- 미국 주식에 투자하는 개인 투자자
- 시장 전체 흐름을 빠르게 파악하고 싶은 트레이더
- 본인 (실제 사용 + 포트폴리오 프로젝트)

### 1.3 핵심 가치
| 기존 도구 | 본 프로젝트의 차별점 |
|---|---|
| finviz | 히트맵 + 스크리너 + 포트폴리오 + 센티먼트를 하나로 통합 |
| Yahoo Finance | LLM 기반 감성분석과 AI 시장 요약 |
| 스프레드시트 | 자동 수익률 계산, 시각화, 배당 캘린더 |

## 2. 기능 요구사항

### 2.1 Feature A: 마켓 히트맵 + 스크리너

#### A-1. 섹터별 트리맵 히트맵
- **설명**: S&P 500 / 전체 시장 종목을 GICS 섹터·산업별로 그룹화하여 트리맵으로 표시
- **유저 스토리**: 투자자로서, 시장 전체의 등락을 한눈에 파악하고 싶다
- **수용 기준**:
  - 트리맵 셀 크기 = 시가총액 비례
  - 셀 색상 = 일간 변동률 (빨강/초록 그라데이션)
  - 섹터/산업 레이블 표시
  - 종목 호버 시 티커, 가격, 변동률, 시가총액 툴팁
  - 섹터 클릭 시 해당 섹터 확대(drill-down)
  - 기간 전환: 1D / 1W / 1M / 3M / YTD / 1Y

#### A-2. 종목 스크리너
- **설명**: 조건 기반 필터로 종목을 검색·정렬
- **유저 스토리**: 투자자로서, 특정 조건에 맞는 종목을 빠르게 찾고 싶다
- **수용 기준**:
  - 필터 조건: 섹터, 산업, 시가총액 범위, P/E, P/B, 배당수익률, 52주 고/저 대비, 거래량
  - 다중 조건 AND 결합
  - 결과 테이블: 정렬 가능한 컬럼 (티커, 이름, 가격, 변동률, 시가총액, P/E, 배당률 등)
  - 페이지네이션 (50개 단위)
  - 결과에서 종목 클릭 → 종목 상세 페이지 이동

#### A-3. 종목 비교
- **설명**: 2~5개 종목을 나란히 비교
- **유저 스토리**: 투자자로서, 관심 종목들의 성과와 밸류에이션을 비교하고 싶다
- **수용 기준**:
  - 종목 검색/추가 (자동완성)
  - 비교 항목: 가격 차트(오버레이), 수익률, 밸류에이션 지표, 재무 지표
  - 정규화 차트 (시작점 = 100%) 지원
  - 비교 테이블 형태로도 확인 가능

#### A-4. 종목 상세 페이지
- **설명**: 개별 종목의 상세 정보 표시
- **유저 스토리**: 투자자로서, 특정 종목의 차트, 재무, 뉴스를 한 곳에서 보고 싶다
- **수용 기준**:
  - 캔들스틱 차트 (1D/1W/1M/3M/6M/1Y/5Y)
  - 거래량 바 차트
  - 기업 프로필 (섹터, 산업, 시가총액, 직원수, 설명)
  - 핵심 재무 지표 (P/E, P/B, EPS, ROE, 부채비율 등)
  - 배당 정보 (수익률, 배당금, 배당일)
  - 최근 뉴스 목록 (센티먼트 태그 포함)

### 2.2 Feature B: 포트폴리오 트래커

#### B-1. 매매 내역 관리
- **설명**: 매수/매도 거래를 수동 입력·관리
- **유저 스토리**: 투자자로서, 내 실제 거래를 기록하고 관리하고 싶다
- **수용 기준**:
  - 거래 입력: 종목(자동완성), 매수/매도, 수량, 단가, 날짜, 수수료
  - 거래 내역 테이블 (수정/삭제 가능)
  - CSV 가져오기/내보내기
  - 다중 포트폴리오 지원 (예: "장기투자", "단기매매")

#### B-2. 포지션 및 손익
- **설명**: 현재 보유 포지션과 수익률 자동 계산
- **유저 스토리**: 투자자로서, 내 보유 종목의 현재 수익률을 한눈에 보고 싶다
- **수용 기준**:
  - 보유 종목 목록: 종목, 수량, 평균단가, 현재가, 평가금액, 수익률(%), 수익금($)
  - 실현 손익 (매도 완료된 거래)
  - 미실현 손익 (현재 보유 중)
  - 총 포트폴리오 가치, 총 수익률, 총 투자금

#### B-3. 자산배분 시각화
- **설명**: 포트폴리오 구성을 시각적으로 표시
- **유저 스토리**: 투자자로서, 내 자산이 어떻게 분산되어 있는지 보고 싶다
- **수용 기준**:
  - 종목별 비중 파이/도넛 차트
  - 섹터별 비중 파이/도넛 차트
  - 포트폴리오 가치 추이 라인 차트 (일별)
  - 벤치마크(S&P 500) 대비 수익률 비교

#### B-4. 배당 캘린더
- **설명**: 보유 종목의 배당 일정 표시
- **유저 스토리**: 투자자로서, 언제 배당금을 받는지 한눈에 보고 싶다
- **수용 기준**:
  - 월별 캘린더 뷰 (배당락일, 지급일 표시)
  - 예상 배당금 합계 (월별/연별)
  - 배당 이력 테이블

#### B-5. 세금 계산 (간이)
- **설명**: 양도소득세 추정 계산
- **유저 스토리**: 투자자로서, 올해 예상 세금을 미리 파악하고 싶다
- **수용 기준**:
  - 실현 손익 기반 양도소득 계산
  - 단기(1년 미만) / 장기(1년 이상) 구분
  - 연도별 요약

### 2.3 Feature C: 마켓 센티먼트 대시보드

#### C-1. 뉴스 감성분석
- **설명**: 최신 시장/종목 뉴스를 LLM으로 감성 분석
- **유저 스토리**: 투자자로서, 뉴스가 긍정적인지 부정적인지 한눈에 파악하고 싶다
- **수용 기준**:
  - 시장 전체 뉴스 수집 (주요 금융 뉴스 소스)
  - 뉴스별 감성 점수 (매우 부정 ~ 매우 긍정, 5단계)
  - 감성 태그 (Bullish/Bearish/Neutral) 컬러 표시
  - 종목별 필터링 가능
  - AI 요약 (3줄 요약)

#### C-2. Fear & Greed 지표
- **설명**: 시장 공포/탐욕 수준을 종합 지표로 표시
- **유저 스토리**: 투자자로서, 현재 시장 심리가 공포인지 탐욕인지 직관적으로 보고 싶다
- **수용 기준**:
  - 게이지 차트 (0~100, Extreme Fear ~ Extreme Greed)
  - 구성 지표:
    - VIX (변동성 지수)
    - 시장 모멘텀 (S&P 500 vs 125일 이동평균)
    - 풋/콜 비율
    - 신고가/신저가 비율
    - 시장 거래량
  - 일별 추이 차트 (30일/90일)
  - 각 구성 지표별 세부 값 표시

#### C-3. 종목별 센티먼트 트렌드
- **설명**: 특정 종목의 감성 변화 추이
- **유저 스토리**: 투자자로서, 관심 종목에 대한 시장 반응 변화를 추적하고 싶다
- **수용 기준**:
  - 종목 검색 → 최근 30일 감성 점수 트렌드 차트
  - 일별 뉴스 건수 + 평균 감성 점수
  - 주요 뉴스 타임라인

#### C-4. AI 시장 리포트
- **설명**: LLM이 당일 시장 상황을 종합 분석·요약
- **유저 스토리**: 투자자로서, 오늘 시장에서 무슨 일이 있었는지 빠르게 파악하고 싶다
- **수용 기준**:
  - 일일 시장 요약 (주요 지수, 섹터 동향, 이슈 종목)
  - 주요 이벤트 하이라이트
  - 시장 전망 (LLM 분석 기반)
  - 수동 새로고침 (API 비용 관리)

## 3. 화면 구성

### 3.1 레이아웃
```
┌─────────────────────────────────────────────┐
│  로고  │  Market  Portfolio  Sentiment  검색  │
├─────────────────────────────────────────────┤
│                                             │
│              메인 콘텐츠 영역                 │
│                                             │
└─────────────────────────────────────────────┘
```

### 3.2 페이지 구조
| 경로 | 페이지 | 주요 기능 |
|---|---|---|
| `/` | 대시보드 홈 | 시장 요약, 주요 지수, 오늘의 센티먼트 |
| `/market` | 마켓 히트맵 | 트리맵 히트맵, 기간 전환 |
| `/screener` | 스크리너 | 필터링, 결과 테이블 |
| `/stock/:ticker` | 종목 상세 | 차트, 재무, 뉴스, 센티먼트 |
| `/compare` | 종목 비교 | 다중 종목 비교 |
| `/portfolio` | 포트폴리오 | 보유 현황, 손익, 자산배분 |
| `/portfolio/trades` | 매매 내역 | 거래 입력/관리 |
| `/portfolio/dividends` | 배당 캘린더 | 배당 일정, 예상 수입 |
| `/portfolio/tax` | 세금 | 양도소득 계산 |
| `/sentiment` | 센티먼트 | Fear & Greed, 뉴스 감성 |
| `/sentiment/report` | AI 리포트 | 일일 시장 요약 |

## 4. 데이터 소스 요구사항

### 4.1 시장 데이터
| 데이터 | 용도 | 후보 소스 |
|---|---|---|
| 주가 (OHLCV) | 차트, 히트맵, 수익률 계산 | yfinance, Alpha Vantage |
| 기업 정보 | 섹터, 산업, 시가총액, 프로필 | yfinance |
| 재무 지표 | P/E, P/B, EPS, ROE 등 | yfinance, FMP |
| 배당 데이터 | 배당 캘린더, 수익률 | yfinance |
| 종목 목록 | 스크리너, 자동완성 | SEC EDGAR, 수동 관리 |

### 4.2 센티먼트 데이터
| 데이터 | 용도 | 후보 소스 |
|---|---|---|
| 뉴스 헤드라인 | 감성분석 입력 | Finnhub News, Alpha Vantage News |
| VIX | Fear & Greed 지표 | yfinance (^VIX) |
| 풋/콜 비율 | Fear & Greed 지표 | CBOE (크롤링) |
| 시장 지표 | 모멘텀, 거래량 등 | yfinance |

### 4.3 LLM
| 용도 | API |
|---|---|
| 뉴스 감성분석 | Anthropic Claude API |
| 시장 요약 리포트 | Anthropic Claude API |

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|---|---|
| 데이터 갱신 | 시장 데이터: 15분~1시간 간격 캐싱, 수동 새로고침 지원 |
| 응답 속도 | 히트맵 초기 로딩 3초 이내, 스크리너 필터 1초 이내 |
| 저장소 | 로컬 DuckDB (서버 불필요, 단일 사용자) |
| 비용 | 무료 데이터 소스 우선, LLM 비용은 수동 트리거로 제어 |
| 브라우저 | Chrome/Edge 최신 버전 |
| 반응형 | 데스크톱 우선 (1280px+), 태블릿 부분 지원 |
